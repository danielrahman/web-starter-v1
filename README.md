# Web Starter

Minimal Next.js app with a single blank homepage that currently renders `START HERE`.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run test:integration`
- `npm run verify:cms`
- `npm run payload:create-admin`
- `npm run payload:push-local-to-remote`
